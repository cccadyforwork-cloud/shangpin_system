# 成功上传模板样板库

这些文件来自桌面 `亚马逊上传表格存档`，保留为已成功上传过的 Amazon 上传表格样板。

用途：

- 对照不同 Product Type 的真实成功字段。
- 提炼字段默认值、条件必填字段和类目差异。
- 给后续自动填表逻辑做回归样本，避免新规则破坏已成功模板。

注意：这些是成功样板，不建议直接覆盖业务项目里的填表版本。新增产品仍应放在 `data/projects/产品项目` 下运行自动填表。
