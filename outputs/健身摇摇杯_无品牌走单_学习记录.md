## 字段坑点

- BOTTLE 模板存在重复列名：
  - `Item Width Unit` 第一次不是 item_dimensions 宽度单位，真正要写的是第二个同名列，脚本映射键为 `Item Width Unit.1`。
  - `Item Height Unit` 第一次不是 item_dimensions 高度单位，真正要写的是第二个同名列，脚本映射键为 `Item Height Unit.1`。
- 自检规则必须按 field_name 找列，不能只按表头文字找第一个同名列。
- 2026-07-12 处理报告新增错误：`90220 - 'Item Height Unit' is required but missing`，受影响单元格为 processing-summary 的 `EV7:EV9`。根因不是后面的 `item_dimensions.height.unit`，而是前面的 `item_width_height.height.unit` 没填。
- BOTTLE 上传前需要同时补齐 `item_width_height` 组：
  - `Item Height Base to Top` = 6.10
  - `Item Height Unit` = Inches
  - `Item Width Top` = 2.95
  - `Item Width Unit` = Inches
- 不要只按 processing-summary 的列字母修，因为 Amazon 返回文件会出现列位置偏移；必须按第 5 行 field_name 精确定位字段。
