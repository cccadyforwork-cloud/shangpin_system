## 核心规则

1. 走无品牌路线。
   - Brand Name 填 `Generic`
   - Manufacturer 填 `Generic`
   - 标题、描述、卖点中不要出现供应商公司名或品牌名，例如 `WESWOO`

2. 三个款式必须作为 3 个单独 SKU 填写。
   - 不填写变体字段
   - 不创建 Parent / Child
   - 不填写 Variation Theme
   - 每行都是独立可售商品

3. SKU 格式为 `Stripper-颜色英文`。
   - 粉色：`Stripper-Pink`
   - 绿色：`Stripper-Green`
   - 红色：`Stripper-Red`
   - SKU 中的颜色必须和对应行的 `Color` 字段一致

4. 尺寸单位必须使用 inches。
   - 原始尺寸：`19 cm x 13 cm x 3 cm`
   - 换算后：`7.48 x 5.12 x 1.18 inches`
   - 所有尺寸单位字段都填 `Inches`
   - 文案中不要保留 `cm` 或 `Centimeters`

5. 价格和重量固定。
   - 售价：`4.99`
   - 商品重量：`106 Grams`
   - 包装重量：`106 Grams`

6. 图片链接未提供时先留空。
   - 不要用竞品图片链接
   - 不要用本地图片路径填 Amazon 图片 URL 字段

## 合规注意事项

1. 这是手动金属园艺工具，不是电动工具。
   - 电池相关字段填 `No` 或留空
   - 不要填写电压、瓦数、电池类型、锂电池参数

2. 不是危险品。
   - `supplier_declared_dg_hz_regulation` 填 `Not Applicable`
   - 不要填写 GHS 分类、Hazmat 编号、SDS URL

3. 不是农药或农药设备。
   - `pesticide_marking.marking_type` 仍需填一个有效值，否则 Amazon 会报 `99022`

   - 农药登记状态填非农药声明
   - 不要填写农药认证编号

4. 无品牌路线下不要出现采购单公司名。
   - 不要填 `WESWOO`
   - 不要填 `WESWOO (CHINA) LIMITED`
   - 不要把供应商名写入标题、品牌、制造商、描述或卖点
