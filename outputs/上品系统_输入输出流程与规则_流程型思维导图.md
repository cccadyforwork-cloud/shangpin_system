# 上品系统输入输出流程与规则 - 流程型思维导图

已生成 XMind 文件：`outputs/上品系统_输入输出流程与规则_流程型思维导图.xmind`

## 主流程

01 建项与资料归位 -> 02 自动提炼资料草稿 -> 03 产品资料上传前自检 -> 04 写入 Amazon 模板 -> 05 模板级自检 -> 06 人工上传与状态记录 -> 07 失败复盘与成功沉淀

## 每个阶段固定结构

- 输入
- 处理
- 输出
- 规则介入

## 反馈闭环

- 模板自检未过：回到模板写入/字段修正
- Amazon 上传失败：解析 processing-summary，修下一版
- 失败复盘和成功样板：沉淀为下一次前置规则

## 规则分类

A 资料与目录规则
B 路线与品牌规则
C Haul 与价格规则
D 尺寸重量规则
E 模板字段规则
F 成功样板规则
G 风险与合规规则
H 上传反馈规则
